About


This is a web interface to query the Universal Natural Products Database (UNPD) originally obtained from http://pkuxxj.pku.edu.cn/UNPD. As the original resource was no longer available, we figured it would be useful to host the data online again. We obtained the data from Pierre Marie Allard's (http://oolonek.github.io/ISDB/). The UNPD structures are freely available and the source code of this webserver are open source under the BSD license. Use it as you please!
